# Recibir Parámetros

Se puede recibir como parámetros valores utilizando $, seguido del número.
Ej:

```bash
if [-d "$1"]
    #codigo
fi
```

# Montar imagenes de disco

Montar

```
sudo mount -t tipoDeParticion nombre.extension directorioDeMontado
```

Desmontar

```
sudo umount directorioDeMontado
```
